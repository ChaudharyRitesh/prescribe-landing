# KaeroPrescribe Landing Page - Clean Design System

## Color Palette (Minimal & Professional)

### Primary Colors

- **Blue**: `#2563eb` (Blue-600) - Main brand color, CTAs, accents
- **White**: `#ffffff` - Backgrounds, text on dark
- **Gray-900**: `#111827` - Dark backgrounds, footer
- **Gray-50**: `#f9fafb` - Light section backgrounds

### Secondary Colors

- **Gray-100**: `#f3f4f6` - Hover states, subtle backgrounds
- **Gray-200**: `#e5e7eb` - Borders, dividers
- **Gray-300**: `#d1d5db` - Disabled states
- **Gray-600**: `#4b5563` - Secondary text
- **Gray-700**: `#374151` - Body text
- **Gray-900**: `#111827` - Headings, primary text

### Accent Colors

- **Green-600**: `#16a34a` - Success, checkmarks
- **Red-600**: `#dc2626` - Errors (if needed)

## Usage Guidelines

### When to Use Blue

- Primary CTAs (Request Demo, Get Started)
- Icons in feature cards
- Accent text in headings
- Active states
- Highlights

### When to Use Gray

- Body text
- Secondary information
- Borders and dividers
- Disabled states
- Backgrounds

### When to Use White

- Main backgrounds
- Card backgrounds
- Text on dark backgrounds

### When to Use Green

- Success states
- Checkmarks in comparisons
- Positive indicators

## Responsive Design

- **Mobile**: Full width, stacked layout
- **Tablet (md)**: 2-column grids, adjusted spacing
- **Desktop (lg)**: 3-4 column grids, full spacing

## Accessibility

- Minimum contrast ratio: 4.5:1 for text
- Blue-600 on white: 8.59:1 ✓
- Gray-600 on white: 7.0:1 ✓
- All interactive elements: 44px minimum touch target
- Focus states: Visible outline or highlight

## Dark Mode (Optional Future)

If implementing dark mode:

- Background: Gray-900
- Text: Gray-50
- Borders: Gray-800
- Blue: Blue-500 (lighter for contrast)
